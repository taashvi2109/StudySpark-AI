# StudySpark AI

StudySpark AI is an AI-powered study assistant that transforms printed notes into structured learning material. By combining OCR with Google's Gemini API, the application extracts text from uploaded documents and generates organized study resources to make revision faster, easier, and more interactive.

## Overview

The application allows users to upload notes in PDF or image format, review the extracted text, and generate multiple AI-powered learning resources from a single document. It is designed to simplify the revision process by converting lengthy notes into concise and well-structured study material.

## Key Features

* Upload notes in PDF, JPG, JPEG, or PNG format
* OCR-based text extraction using EasyOCR
* Editable preview before AI processing
* AI-generated summaries
* Automatic key point extraction
* Difficulty-based practice MCQs
* Flashcards for quick revision
* Easy explanations for better understanding
* Download generated study material as a formatted PDF

## Application Workflow

### Home Page

Upload handwritten or printed notes through a simple and user-friendly interface.

### OCR Text Extraction

The uploaded document is processed using OCR, and the extracted text is displayed for review. Users can edit the extracted content before generating study material.

### Study Material Generation

Select the required study resources, including AI Summary, Key Points, Practice MCQs, Flashcards, and Easy Explanation.

### AI Summary

A concise summary highlighting the most important concepts from the uploaded notes.

### Key Points

Important concepts are extracted into well-structured bullet points for quick revision.

### Practice MCQs

Automatically generated multiple-choice questions help reinforce learning and assess understanding.

### Flashcards

Interactive flashcards support active recall and efficient revision.

### Easy Explanation

Complex concepts are rewritten in simple, easy-to-understand language to improve clarity.

### PDF Export

The complete AI-generated study material can be downloaded as a neatly formatted PDF for offline access.
